## Termal commands.
Note : ($ means term prompt):
```bash
$ ls            # List folder conent
$ cd <dest>     # Chang dir
$ cd ..         # Move to parent dir
$ pwd           # Working Dir
```

.               # Current Directory

## g++
```bash
# To compile
$ g++ <source name>.cpp
# -o for output name.
$ g++ <Source name>.cpp - o <output name>
$ .\<output name>.exe
```
